Binary Number Calculator

This calculator takes binary numbers as input, converts them to decimal, evaluates the expression, then returns the result as a binary number. For Example: The number 9 is represented as "1001" in binary. The input: "1001+1001" in this calculator is the equivalent to entering "9+9" in a traditional calculator. The calculation is performed when the equals button is clicked (9+9=18), finally the result is returned back in binary form ('10010' in binary - which is the same as the number 18).
